Two Factor Authentication Service

 

Sends a text message to a given destination and returnd a Job ID.
This JobID together with a token can be validated and will then return either "true" or "false"

Usage:


http://localhost:8080/send
Generate Message and Send Token
Query Parameters:
localhost:8080/send?dest=017615528046&length=5&type=numbers&exp=1h&msg="Bernhard, Dein Token: !TOKEN!. Tipp ihn ein!"
dest: mobile phone number incl intl dialling prefix
exp: token expiry time in duration notation (e.g. 5m or 1h)
length: token length (int)
msg: text message to be sent - placeholder for the token is !TOKEN!
type: token type - numbers, string, lstring, ustring, symbol

http://localhost:8080/check
Check Validity
